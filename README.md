## Projects

**[Identify.AI](https://github.com/ThomasPaulCJ)** — Forensic sketch generator  
Fine-tuned Stable Diffusion v1.5 to generate facial composites from natural language descriptions. Flask backend, React frontend. Won 1st place at VISAT Dhruva'25 EmergeX Expo.  
`Stable Diffusion` `Flask` `React` `Computer Vision`

<br/>

**[Calm.ai](https://github.com/ThomasPaulCJ)** — Mental wellness companion  
LLM agent that reads journal entries, reads the mood, and suggests Spotify playlists and exercises. Built with LangGraph for the agent flow.  
`LangGraph` `Flask` `React` `Spotify API`

<br/>

**[Air Dash](https://github.com/ThomasPaulCJ)** — Gesture controlled game  
2D Unity game controlled entirely by hand gestures via webcam. MediaPipe detects gestures in Python and sends inputs to the game over a socket.  
`Unity 2D` `C#` `Python` `MediaPipe` `OpenCV`

<br/>

**[Krakens for a Greener Future](https://github.com/ThomasPaulCJ/Krakens-For-A-Greener-Future)** — Climate awareness platform  
Web platform built around SDG 13, 14 and 15 with educational content and community features.  
`HTML` `CSS` `JS`
